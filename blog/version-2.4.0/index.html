Motion Canvas v2.4.0

February 18, 2023 · One min read
Jacob
Motion Canvas Creator

Features​

  • DEFAULT signal values. lineHeight now supports percentages. (#259) (18f61a6)
  • New moveBelow, moveAbove and moveTo methods for rearranging the nodes. (#365) (16752a3)
  • New CodeBlock documentation (#302) (73f7221)
  • The editor now warns about duplicated time events. (#341) (053b2a6)
  • We now have a search bar on the documentation website. (#335) (48f74a6)
  • You can now open the documentation from within the editor. (#346) (fc4ee5d)
  • SHIFT + -> moves to the last frame. (#354) (4b81709), closes #353
  • Flexbox gap is now represented by a Vector2 signal. (#355) (3cae97e), closes #352
  • New Matrix2D type. (#340) (66b41e6)

Bug Fixes​

  • 2d: fix Gradient and Pattern signals (#376) (6e0dc8a)
  • 2d: fix layout calculation for nodes not explicitly added to view (#331) (528e2d5)
  • 2d: format whitespaces according to HTML (#372) (83fb565), closes #370
  • core: playback speed is reset after saving with faulty code (#204). (#339) (6771e5e)
  • docs: fix search (#336) (e44ec02)
  • typo on codeblock remove comments (#368) (2025adc)
  • ui: fix inspector tab (#374) (c4cb378)
  • vite-plugin: fix js template (#337) (3b33d73)
  • vite-plugin: ignore query param in devserver (#351) (5644d72)
